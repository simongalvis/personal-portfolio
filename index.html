Some Of My Projects:

All-In Recruits

All-In Recruits is an application made for a client that was seeking to improve their talent-acquisition process. The application provides a centralized location for company administrators to view and manage applicant submissions. All-In Recruits also provides job seekers with a platform to submit their resume and information for a chance to be considered as a candidate for All-In Education's next available positions. Check out the Github Repo for more information.

Breakaway

Breakaway is an application made for the sports enthusiast or just anybody seeking to track their athletic progress. Breakaway provides its users with a platform to log their daily exercise activities in one convenient location. Users can create an account, log activities, and view their previous activities on subsequent login attempts. Check out the Github Repo for more information.

EduFind

EduFind is a school search tool that allows users to access a comprehensive list of schools based on a set of parameters. Users can look up schools of their interest based on their input of location and school name. DOM traversal and manipulation, GET requests, event handling, and promises were used to implement this application. Check out the Github Repo for more information.
